## 🏗️ Architecture
## 🛠️ Tech Stack
- **Source:** GitHub
- **Pipeline:** AWS CodePipeline
- **Build:** AWS CodeBuild
- **Deploy:** AWS CodeDeploy
- **Servers:** AWS EC2 (Amazon Linux 2023)
- **Web Server:** Apache HTTPD

## 📋 Pipeline Flow
1. Push code to GitHub main branch
2. CodePipeline automatically triggers
3. CodeBuild compiles and packages the app
4. CodeDeploy deploys to EC2 in both regions
5. Website live in minutes!
